IntraMaps Roam

Simple and flexible data collection built using QGIS.

Download this project as a .zip file Download this project as a tar.gz file

IntraMaps Roam is a field, and office, data collection application built using the QGIS libraries and Python. It was built to be easy to use on (windows) based devices.

See the wiki for information on how to create a project and user guides.

The latest release is V2.1 - Data Collection

Newest features in 2.1 include:

  • Polygon and line capture support
  • Recapture geometry
  • Drawing pad
  • Camera (Windows only)
  • GPS page
  • Translation support
  • API clean up and extras (access to GPS on forms etc)
  • Legend page
  • Attribute Help
  • GPS Logging and tracking
  • New roam.api module for public facing API. Anything imported from this module will have stable API in 2.x releases.
  • Loading projects from other folders

More details can be found on the features page for the release.
